Commands

Movement:

  • FORWARD n (or FD n) - Move forward n pixels
  • BACK n (or BK n) - Move backward n pixels
  • RIGHT n (or RT n) - Turn right n degrees
  • LEFT n (or LT n) - Turn left n degrees

Pen Control:

  • PENUP (or PU) - Lift pen (don't draw)
  • PENDOWN (or PD) - Put pen down (draw)
  • SETPENCOLOR n - Set pen color (0-15)

Screen:

  • HOME - Return to center
  • CLEARSCREEN (or CS) - Clear screen
  • HIDETURTLE (or HT) - Hide turtle
  • SHOWTURTLE (or ST) - Show turtle

Control:

  • REPEAT n [commands] - Repeat commands n times
